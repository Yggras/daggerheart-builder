# Glossary

## Canonical Data

Structured SRD data that has been reviewed and accepted as safe for app use.

## Compendium

The searchable rules reference containing SRD entries such as spells, loot, equipment, domains, classes, subclasses, ancestries, communities, adversaries, environments, and other rules content.

## Daggerheart SRD

The official Daggerheart System Reference Document used as the project's rules data source.

## DM View

Campaign-facing tools and views intended for the game master.

## Extracted Data

Raw or partially processed data produced from the SRD PDF. Extracted data is untrusted until reviewed.

## Offline-First

The app should continue to provide useful functionality without network access wherever feasible, using local data and later synchronization.

## Playable Campaign View

The live play experience that includes character sheets, shared state, DM tools, and real-time client updates.

## Reviewed Data

Extracted SRD data that has been manually checked and corrected against the official source.

## Second Brain

The repo-local project memory in `docs/brain/`.
